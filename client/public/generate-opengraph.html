OpenGraph Image Generator

Hướng dẫn tạo OpenGraph Image:
  1. Mở file này trong trình duyệt
  2. Scroll xuống phần preview bên dưới
  3. Chụp màn hình phần preview (kích thước 1200x630px)
  4. Hoặc sử dụng công cụ online như Canva hoặc Figma
  5. Lưu file với tên opengraph.jpg vào thư mục client/public/
Lưu ý: Kích thước chuẩn cho OpenGraph image là 1200x630px. Sau khi tạo xong, xóa file HTML này và deploy lại để cập nhật.
